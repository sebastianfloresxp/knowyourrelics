# Catholic Relics Directory

Standalone static website for browsing a large directory of Catholic relics, shrine treasuries, and public relic collections.

## Open It

You can open the site directly in a browser:

- `index.html`

If you prefer a local server:

```bash
cd catholic-relics-atlas
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Files

- `index.html`: homepage
- `atlas.html`: searchable relic directory
- `data.js`: relic dataset and featured collection blocks
- `main.js`: homepage rendering
- `guide.js`: directory filters and detail panel
- `styles.css`: visual design
