How the directory writes each entry.

Where it is

Each record states the church, cathedral, basilica, chapel, shrine, museum, treasury, or inventory site where the relic is listed.

How to see it

Entries explain whether the item is visible in an ordinary shrine setting, part of a permanent collection, or only published in a custody list.

Public display
